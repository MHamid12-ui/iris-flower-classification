# 🌸 Iris Flower Classification

Machine Learning model to classify Iris flowers into 3 species based on petal and sepal measurements.

## 🌼 Species
- Setosa
- Versicolor
- Virginica

## 🏆 Results
| Model | Accuracy |
|-------|----------|
| Gradient Boosting ⭐ | 96.67% |
| Support Vector Machine | 96.67% |
| Logistic Regression | 93.33% |
| K-Nearest Neighbors | 93.33% |
| Decision Tree | 93.33% |
| Random Forest | 90.00% |

**Best Accuracy: 96.67%**

## 📊 Features Used
- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

## 🛠️ Libraries Used
- Python, Scikit-Learn, Pandas, NumPy
- Matplotlib, Seaborn
